========================================
TROUBLESHOOTING / PORT CLEANUP:
========================================

If ports are stuck/in use, run this BEFORE starting servers:

```powershell
# Kill all processes on target ports
Get-NetTCPConnection -LocalPort 8016 -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
Get-NetTCPConnection -LocalPort 8091 -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
Get-NetTCPConnection -LocalPort 8015 -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
Get-NetTCPConnection -LocalPort 8000 -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue

# Or force-kill by PID if above doesn't work:
taskkill /PID <PID> /F

# Verify ports are clean:
netstat -ano | Select-String ':80(0|15|16|91)'
```
